## 关于网站

>本仓库 Fork 自 [yihong0618/gitblog](https://github.com/yihong0618/gitblog)，一个基于 GitHub Issues 和 Actions 构建的个人博客系统。
🙏 感谢 @yihong0618 打造的优秀项目！

作为一位计科学生，从大学开始便开始与计算机和编程语言打交道，本科主要学习语言是Java，于是假期就在B站看视频，学习Java相关知识，然后老师建议要做笔记，要开始写博客，学Java的课程第一节就教Markdown，以及注册CSDN和博客园。于是便接触到搭建博客、网站这些东西，想着身为计科学生，如果可以搭建一个自己的网站记录学习情况和笔记肯定很酷，这可能就是这个网站的来源。

大二时，模仿B站大佬写的博客，一个由bootstrap为前端，Java为后端的简约博客，但是需要服务器部署才行，于是便开始寻找免费的方案。在B站上看到各种使用Hexo的快速建站视频吸引了我的注意力，便开始用Hexo搭建网站，搭配各种主题，很炫酷，为此没忍住花钱买了阿里云的域名。寒假开学后，沉迷于打游戏、看动漫、看漫画，于是网站就废了...一气之下，删除了所有网站源码。

现在作为研究生（混上了），每天在实验室看论文、做笔记，于是又萌发了搭建网站记笔记的念头，于是又开始搭建网站了，便开始了使用Hugo建站。由于使用hugo不能随时随地修改笔记，而且hugo还是比较笨重的，因此转为直接使用GitHub的issue来构建简单的博客系统。希望这次能坚持做笔记...
